Add / Edit State

This screen displays a County and it's properties.

   
 

State/Province Code for Order Reporting
Code: Enter the value that will be saved and exported with the Order data if this State is selected.

For example: "TX".
 

State Name
Name: Enter the name for this State that will be displayed to the Customer on any billing or shipping address entry field.

For example: "Texas".
 

Country
Country: Select any existing Country that this State is geographically located within.

For example: "United States".

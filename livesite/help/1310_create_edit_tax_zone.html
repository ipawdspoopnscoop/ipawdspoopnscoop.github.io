Create / Edit Tax Zone

This screen displays a Tax Zone's area and tax rate.

   
 

Tax Zone Name
Tax Rate Name: This is the name for the Tax Zone.
Tax Rate (%): This is the percentage of the tax to apply to each Product item.
 

Tax Zone Countries
Allowed Countries: Select the Countries that this Tax Zone will apply too.
Disallowed Countries: Select the Countries that this Tax Zone will never apply too.
 

Tax Zone States/Provinces
Allowed States: Select the States/Provinces that this Tax Zone will apply too.
Disallowed States: Select the States/Provinces that this Tax Zone will never apply too.
